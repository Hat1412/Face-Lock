# Face_Lock
Face Recognition with Python

To use this program:

<strong> <h1> Run the following commands in cmd </h1> </strong>

To download the files:
  ```
  git clone https://github.com/Hat1412/Face-Lock.git
  cd Face-Lock
  ```
To run file:
  ```
  #In windows:
  python main.py
  #In Linux/MacOS
  python3 main.py
  ```
